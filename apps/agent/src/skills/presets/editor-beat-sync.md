# Beat-Sync Editing

## Rules
- Cut on the beat when BPM > 100
- Use visual transitions on half-beats
- Maintain minimum 2-frame hold between cuts
- Align clip starts to downbeats for high-energy sequences
- Use slow dissolves on sections where BPM drops below 80
