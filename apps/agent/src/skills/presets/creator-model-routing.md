# Model Routing for Creator Tasks

## Rules
- Use fast models for ideation and brainstorming passes
- Route long-form scripting to higher-capacity models
- Use vision-capable models when analyzing reference imagery
- Prefer lower-latency models for real-time interactive sessions
- Always log model choice and reasoning for post-analysis
