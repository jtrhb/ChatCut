# Prompt Engineering for Content Creation

## Rules
- Always specify tone, audience, and platform in generation prompts
- Use chain-of-thought instructions for complex narrative structures
- Include negative constraints to avoid generic outputs
- Seed prompts with brand voice examples when brand context is available
- Request multiple variants (at least 3) for A/B testing opportunities
